Entities

Entities are special Javascript objects which are entries of the Entities array. The engine includes a pre-defined set of entity types. A developer can add extra entity types by modifying the engine code, but pixlwrld maker strives to have enough entity types available to serve most needs.

Creating Entities

.ent create <type> <arg(s)...>

This command creates an entity and adds it to the global Entities list for .process()ing and .draw()ing. It will also switch the focus to that new entity so that further .ent commands such as .ent mod will refer to the new entity.

Each entity type has a set of required arguments that must be passed to the command. In addition to these required fields, each entity type has a set of optional fields. Including these fields in the command is optional. Each optional field has a default value defined by the entity type which it will be initialized to if not specified.

.ent create npc 4 10 10 16 16 visible false

The first 5 fields (starting with "4") are the required arguments to create an npc. After we've passed those arguments we give the name of another property "visible" and it's value "false". You can pass as many extra arguments as you'd like to. Attempting to modify a field that does not exist in that entity type or attempting to set a field to an incorrect data type will cause nothing to happen. You should not be able to break the program this way.

"Focused" Entity

.ent focus <id> selects an entity from the Entities list. Besides .ent create, all the .ent commands operate on the focused entity.

Modifying Entities

.ent mod <args...> This command can be used to modify the properties of the currently focused entity. You can modify as many properties as you'd like at one time with this command. e.g. .ent mod visible true enabled false

Deleting Entities

.ent delete removes an entity from the Entities list and shifts subsequent entries in the list to fill the gap.
