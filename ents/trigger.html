Entity Type: Trigger

A trigger entity is invisible and runs a command whenever a player enters it.

Initialization Format

To create a trigger entity, use the following command:

.ent create trigger <x> <y> <w> <h> <cmd>

Properties

Property Type Default Description
visible boolean true
enabled boolean true
x integer N/A: Required x position
y integer N/A: Required y position
w integer N/A: Required width
h integer N/A: Required height
cmd string N/A: Required command to run
intersecting bool false
