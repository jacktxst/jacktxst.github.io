Entity Type: NPC

An NPC displays a sprite in the world and can be easily animated via commands or scripts that modify the entity's properties. Besides display, NPCs have no other function, but they can be used together with other entities like interact to create complex systems.

Initialization Format

To create an NPC entity, use the following command:

.ent create npc <spr> <x> <y> <w> <h>

Example Usage

Create an NPC and then add interaction behavior:

.ent create npc 1 100 100 32 32
.ent create interact 100 100 32 32 ".exec create-dialog-box"
