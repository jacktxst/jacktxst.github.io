Entity Type: Interact

Interact entities create interactive areas in the world that players can interact with. These are typically used for dialogue, item collection, or triggering events when the player approaches or clicks on them.

Initialization Format

To create an interact entity, use the following command:

.ent create interact <x> <y> <w> <h> <cmd>

Properties

Property Type Default Description
visible boolean true
enabled boolean true
x integer N/A: Required
y integer N/A: Required
w integer N/A: Required
h integer N/A: Required
cmd string N/A: Required

Usage Examples

.ent create interact 347 1045 16 16 ".exec frog_dialog"
.edit frog_dialog
.prevent interact
#...scripted sequence, create dialog boxes, etc.
.end
.edit frog_dialog_exit
.allow interact
#...misc cleanup
.end

If the player interacts with the interact entity and the script creates a dialog box, the interact entity will still accept interactions from the player, leading to major issues like the creation of duplicate dialog boxes. In most cases, the script called by an interact should contain the line .prevent interact to stop the player from retriggering an already ongoing scripted sequence. Make sure to run .allow interact once the dialog is done.
