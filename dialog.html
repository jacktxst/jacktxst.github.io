The commands starting with .db are used to manage dialog boxes. In a similar fashion to entities, dialog boxes have a specific set of typed properties.

Property Type Default Description
visible boolean true
enabled boolean true
x integer N/A: Required
y integer N/A: Required
w integer N/A: Required
h integer N/A: Required
cols integer
rows integer
panx integer
pany integer
lines string array

Dialog Commands (.db)

command syntax description
.db new <x> <y> <w> <h>creates an empty box
.db focus <id>
.db editallow the user to use the box as a text editor
.db choice <up> <left> <down> <right> <script_name>add a menu choice to the dialog box at the position where the cursor currently is. the direction arguments are indexes of other choices that should be jumped to when those arrow keys are pressed and script_name runs whenever this choice is selected and the confirm key is pressed..
.db delete
.db toscript <name>saves a script which will recreate this dialog box when called
.db s <name>saves the text content of this dialog box directly to a script for in-game script editing purposes
.db write <line>write a single line of text to the box
.db mode <overtype|menu>determines the behavior of the cursor
.db cursor <col> <row>sets the position of the cursor
.db playplay a character-by-character text printing animation
.db x <x>
.db y <y>
.db w <w>
.db h <h>
